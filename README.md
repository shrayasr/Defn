A simple static site that generates dictionary-style definitions for made-up words. No build step, no dependencies to install â€” just open the HTML file.

## Features

- Classic dictionary styling with sepia theme
- Multiple definitions and example usage
- Shareable URLs (compressed, so they're not too long)
- Copy as image for easy sharing
- Mobile friendly

## Usage

1. Open `_defn.html` in your browser
2. Enter your word, type, definitions, and examples
3. Click Generate
4. Share the URL or copy as image

## Hosting

Drop `_defn.html` on any static host. That's it.
